# Gestão Proff

## Descrição
O Gestão Proff é um sistema de gestão de instituições de ensino, projetado para oferecer uma abordagem intuitiva e
eficiente na administração acadêmica. Ele visa simplificar a gestão escolar, unindo praticidade e inovação.

## Como Usar
1. **Clone o Repositório:**
      git clone https://github.com/MatheusChavesf/GestaoParaIES.git
      cd GestaoParaIES

   Funcionalidades
Alunos
Editar: Permite editar as informações de um aluno.
Remover: Remove um aluno do sistema.
Detalhes: Exibe informações detalhadas sobre um aluno.
Professores
Editar: Permite editar as informações de um professor.
Remover: Remove um professor do sistema.
Detalhes: Exibe informações detalhadas sobre um professor.
Disciplinas
Editar: Permite editar as informações de uma disciplina.
Remover: Remove uma disciplina do sistema.
Detalhes: Exibe informações detalhadas sobre uma disciplina.
Cursos
Editar: Permite editar as informações de um curso.
Remover: Remove um curso do sistema.
Detalhes: Exibe informações detalhadas sobre um curso.
Avaliações
Editar: Permite editar as informações de uma avaliação.
Remover: Remove uma avaliação do sistema.
Detalhes: Exibe informações detalhadas sobre uma avaliação.
Certificados
Editar: Permite editar as informações de um certificado.
Remover: Remove um certificado do sistema.
Detalhes: Exibe informações detalhadas sobre um certificado.
Mensalidades
Editar: Permite editar as informações de uma mensalidade.
Remover: Remove uma mensalidade do sistema.
Detalhes: Exibe informações detalhadas sobre uma mensalidade.
Presença
Editar: Permite editar as informações de presença de um aluno.
Remover: Remove registros de presença do sistema.
Detalhes: Exibe informações detalhadas sobre a presença de um aluno.
Turma
Editar: Permite editar as informações de uma turma.
Remover: Remove uma turma do sistema.
Detalhes: Exibe informações detalhadas sobre uma turma.
